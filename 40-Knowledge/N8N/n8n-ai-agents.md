## O que são AI Agents?

Um AI agent é um sistema autônomo que:
- Recebe dados
- Toma decisões racionais
- Atua no ambiente
- Alcança objetivos específicos

---

## Componentes Principais

### 1. Sensores (Sensors)
- Recebem dados do ambiente
- Inputs do usuário, APIs, databases

### 2. Atuadores (Actuators)
- Executam ações
- Enviar emails, criar registros, chamar APIs

### 3. Motor de Reasoning (Brain)
- Toma decisões baseadas no ambiente
- Usa LLM para processar e decidir
