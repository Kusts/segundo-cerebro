# Model Context Protocol (MCP) - Overview

> **Fonte:** https://modelcontextprotocol.io/introduction
> **Data:** 2026-03-13

---

## O que é MCP

**Model Context Protocol** é um padrão open-source para conectar aplicações de IA a sistemas externos.

Think of MCP like a **USB-C port for AI applications** — standardized way to connect AI apps to external systems.

---

## O que MCP permite

- Agentes acessam Google Calendar e Notion
- Claude Code gera apps web usando designs do Figma
- Chatbots empresariais conectam a múltiplos bancos de dados
- Modelos de IA criam designs 3D no Blender

---

## Por que MCP importa

### Desenvolvedores
- Reduz tempo de desenvolvimento
- Menos complexidade ao integrar com IA

### Aplicações de IA
- Acesso a ecossistema de data sources, tools, apps
- Melhora experiência do usuário

### Usuários Finais
- Aplicações de IA mais capazes
- Podem acessar dados e executar ações

---

## Suporte ao Ecossistema

MCP é suportado por:
- **Claude** (Anthropic)
- **ChatGPT** (OpenAI)
- **VS Code**
- **Cursor**
- E muitos outros...

---

## Como começar

1. **Build servers** - Criar servidores MCP para expor dados e ferramentas
2. **Build clients** - Desenvolver apps que conectam a servidores MCP
3. **Build MCP Apps** - Criar apps interativos que rodam dentro de clientes de IA
