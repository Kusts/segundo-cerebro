## 1. Pai Rico, Pai Pobre (Robert Kiyosaki)

### Conceitos-Chave

**A diferença entre ativos e passivos:**
- Ativo = algo que põe dinheiro no seu bolso
- Passivo = algo que tira dinheiro do seu bolso

**Os 4 quadrantes:**

```
              E (Empregado)          B (Dono de negócio)
    +------------------------+------------------------+
    |                        |                        |
    | Trabalha pelo dinero  | Sistema trabalha       |
    |                        | para você              |
    +------------------------+------------------------+
    |                        |                        |
    | Trabalha pelo dinero  | Investimentos          |
    | (autoemprego)         | geram renda            |
    +------------------------+------------------------+
    S (Especialista)        I (Investidor)
```

**Lições principais:**
1. Não trabalhar por dinheiro - fazer o dinheiro trabalhar para você
2. Investir em imóveis primeiro (cashflow)
3. Cuidar do próprio negócio (seu patrimônio)
4. Trabalhar para aprender, não só para ganar

## 2. Enriquecer (Lair Ribeiro)

1. Física (saúde)
2. Mental (conhecimento)

4   Espiritual (propósito)

- Riqueza = mau
- Dinheiro = difícil de ganhar
- Rico = gente má

vs.

- Dinheiro = ferramenta de liberdade
- Eu mereço enriquecer
- Rico = alguém que serve

- Leitura diária
- Exercício físico

## 3. Os Segredos da Mente Milionária (T. Harv Eker)

### Conceitos-Chave

**As 17 arquivos da riqueza:**

1. **Abundância vs. Escassez**
   - Ricos: "Há muito dinheiro para todos"
   - Pobres: "Dinheiro é limitado"

2. **Hierarquia de pensamento:**
   - Resultado → Ação → Capacidades → Estado → Identidade

3. **Comportamentos do rico:**
   - Criam múltiplas fontes de renda
   - Investem em educação
   - Trabalham para ganhos passivos
   - Cercam-se de pessoas de sucesso

4. **Arquivos financeiros:**
   - Conta de operações
   - Conta de резерв (10%)
   - Conta de educação (10%)
