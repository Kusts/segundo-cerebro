### O que foi descoberto

- **1.000+ fluxos prontos** no Google Drive
- Templates por nicho: Odontologia, Imobiliária, Saúde
- Self-hosted = mais controle + economia

### Nicho inexplorado: ODONTOLOGIA

- Apenas ~5 fluxos específicos
- Oportunidade: Consultório + Chatbot + WhatsApp
- Fluxos encontrados:
  - `48 - Clinica odontologica.json`
  - `33 - Consultorio - clinica.json`

### Como monetizar Typebot

1. Baixar JSON do Typebot
2. Importar no Typebot.io
3. Personalizar com dados do cliente
4. Conectar WhatsApp (Evolution API)
5. Vender como projeto completo

## 🎨 VIBE CODING - METODOLOGIA

### Fluxo do Vibe Coder

```
IDEA → SPEC → VIBE → IA GENERATES → REVIEW → DEPLOY
```

### Conceitos Essenciais

| Conceito | Descrição |
|----------|-----------|
| **Auth** | Quem é você? (Login, JWT, OAuth) |
| **Authorization** | O que pode fazer? (Roles, RLS) |
| **Sync** | Espera terminar |
| **Async** | Dispara e continua |
| **State** | Local (useState) vs Global (Context) |

### Erros Comuns

1. ❌ Não entender o código gerado
2. ❌ Ignorar segurança
3. ❌ Tudo de uma vez (mega prompts)

### Solução

- ✅ Especificar detalhadamente
- ✅ Pequenos passos iterativos
- ✅ Sempre incluir segurança

## 🧠 REGRAS DE OURO

### Para Vender IA

1. **Não venda tecnologia** - venda resultado
2. **80% do tempo em vendas** - regra 80/20
3. **Follow-up é tudo** - acompanhamento pós-venda
4. **Cliente ruim = dor de cabeça** - seja seletivo
5. **Contratos** - sempre proteja-se

### Para Construir

1. **Especifique claro** - detalhes > vagueza
2. **Pequenos passos** - iteração > big bang
3. **Segurança primeiro** - auth + authorization
4. **Teste sempre** - não assuma que funciona

### Para Precificar

1. **Mínimo R$1.500** por projeto
2. **50% antes** - financeiro
3. **Não faça sozinho** - contrate quando puder
