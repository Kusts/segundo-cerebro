# Supabase - The Postgres Development Platform

## O que aprendi

### Overview
Supabase é uma plataforma de desenvolvimento Postgres completa. Fornece um banco de dados Postgres completo para cada projeto com funcionalidades em tempo real, backups, extensões e mais.

### Produtos/Features

| Produto | Descrição |
|---------|-----------|
| **Database** | Postgres completo com Realtime, backups, extensões |
| **Auth** | Email/password, passwordless, OAuth, mobile logins |
| **Storage** | Armazenar, organizar, transformar arquivos |
| **Realtime** | Listen to DB changes, sync estados entre clients |
| **Edge Functions** | Server-side functions distribuídas globalmente |

### Módulos Postgres
Supabase suporta migração de:
- Amazon RDS
- Auth0
- Firebase Auth/Storage
- Firestore
- Heroku
- MSSQL
- MySQL
- Neon
- Postgres
- Render
- Vercel Postgres

### Recursos Adicionais

- **Management API** - Gerenciar projetos e organizações
- **Supabase CLI** - Desenvolver, gerenciar e fazer deploy
- **Platform Guides** - Ferramentas e serviços
- **Integrations** - Integrações com parceiros
- **Supabase UI** - Componentes pré-construídos

### Características

- **Open Source** - Construído abertamente desde o dia 1
- **Self-hosted option** - Você pode hospedar sua própria instância
- **Scale** - "Build in a weekend, scale to millions"
- **Enterprise ready** - Segurança e governança
