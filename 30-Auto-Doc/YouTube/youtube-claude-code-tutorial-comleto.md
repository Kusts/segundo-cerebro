É uma ferramenta de IA para programação que ajuda desenvolvedores a criar projetos de forma mais rápida e eficiente. Funciona como um "copiloto" avançado que pode entender código, criar arquivos, e executar tarefas de desenvolvimento.

1. **Fundamentos do Claude Code** - O que é e para que serve

3. **Principais recursos** - Funcionalidades principais
4. **Integração com IDE** - Como usar sem precisar de terminal
5. **Do Zero ao Projeto** - Criar um projeto completo na prática

### Para quem é?
