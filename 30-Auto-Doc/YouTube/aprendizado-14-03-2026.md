## Principais Insights

### 🤖 O que são AI Agents?
1. **Nível 1 - LLMs**: Apenas respondem, sem memória
2. **Nível 2 - AI Workflows**: Automação com passos definidos
3. **Nível 3 - AI Agents**: Tomam decisões autónomas, usam ferramentas

### 🛠️ Componentes de um Agent (n8n)
- **Trigger**: O que inicia (schedule, webhook, chat)
- **Chat Model**: O cérebro (OpenAI, Claude, etc)
- **Memory**: Lembra conversas anteriores
- **Tools**: Ações que o agent pode executar

### 💡 Práticas Recomendadas
1. **Self-hosting** > Cloud para agents:
   - Sem limites de execução
   - Mais estável e barato a longo prazo
   - Controle total dos dados

2. **Arquitetura determinística**:
   - Separa lógica de decisão (LLM) da execução (scripts)
   - 90% de precisão por etapa = 59% após 5 etapas
   - Empurre complexidade para código determinístico

### 🎯 Exemplo Prático
Agent de newsletter automática:
- Coleta dados do mercado (S&P 500)
- Formata em Markdown → HTML
- Envia por Gmail automaticamente

### 🔗 Tecnologias Mencionadas
- n8n (automação)
- OpenAI API
- Claude API
- BridgeSpace/BridgeVoice (vibe coding)
- Hostinger VPS (self-hosting)
