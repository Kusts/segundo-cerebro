# GitHub Copilot - AI Pair Programmer

## O que aprendi

### Overview
GitHub Copilot é o "AI pair programmer" do GitHub. Disponível em várias modalidades.

### Produtos

| Produto | Descrição |
|---------|-----------|
| **Copilot in VS Code** | No seu editor |
| **Copilot CLI** | Na linha de comando |
| **Copilot Agents** | Agentes que criam PRs automaticamente |
| **Copilot for Business** | Para empresas |

### Features Principais

1. **Multiple LLMs** - Escolha entre modelos otimizados para velocidade, precisão ou custo
2. **Custom Agents** - Use seus próprios agentes ou de terceiros
3. **MCP Support** - Integração com MCP servers
4. **Copilot Spaces** - Crie base de conhecimento para times

### Preços

| Plano | Preço |
|-------|-------|
| **Free** | $0 |
| **Pro** | $10/mês ou $100/ano |
| **Pro+** | $39/mês ou $390/ano |
